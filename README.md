¿Qué es un programa?
Un programa es un conjunto de instrucciones que le indican a una computadora cómo realizar una tarea específica. Es como una receta que la computadora sigue para lograr un objetivo.
Los programas están escritos en lenguajes de programación, que son lenguajes formales diseñados para ser entendidos tanto por humanos como por máquinas.
Variables:
Una variable es un nombre que se le da a una ubicación en la memoria de la computadora donde se almacena un valor. Es como una etiqueta que permite acceder y modificar ese valor.
Las variables tienen un tipo de dato asociado (por ejemplo, entero, decimal, texto) que indica qué tipo de valor pueden almacenar.

```python
# Un programa simple para calcular el área de un rectángulo


base = 10  
altura = 5 

   (multiplicación: *)
area = base * altura


es_cuadrado = (base == altura)

print("--- Calculadora de Área de Rectángulo ---")
print(f"La base es: {base}")
print(f"La altura es: {altura}")
print(f"El área del rectángulo es: {area}") 
if es_cuadrado:
    print("¡Es un cuadrado porque la base es igual a la altura!")
else:
    print("No es un cuadrado.")
```
